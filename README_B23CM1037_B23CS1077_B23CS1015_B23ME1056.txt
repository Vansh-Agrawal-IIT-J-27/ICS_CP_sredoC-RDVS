HOSPITAL MANAGEMENT SYSTEM
The Hospital Management System is a console-based application designed to manage patient records, allowing for efficient storage, retrieval, and modification of data related to hospital operations. This system enables users to perform various functionalities like patient registration, search for patient details, edit and delete records, and view existing records.

Features
Patient Records Management
Patient Registration: Capture and store patient details including name, age, gender, contact information, and medical record numbers.
Inpatient and Outpatient Records: Maintain records for both inpatients and outpatients, tracking admission and discharge dates.
Doctor-Patient Association
Associate Doctors: Associate and retrieve information about doctors assigned to each patient.
Search Functionality
Search by Multiple Criteria: Enables users to search for patients based on their name, medical record number, or mobile number.
Getting Started
Prerequisites
This program is written in C and requires a C compiler to build and run the application.

Usage
The system provides a menu-driven interface for users to interact with the functionalities:

1) New Data: Add new patient records to the system.
2) Search: Search for patient records by name, medical record number, or mobile number.
3) Edit Data: Modify existing patient records based on their medical record number.
4) Delete Data: Remove patient records from the system using their medical record number.
5) View Data: Display all stored patient records.

File Management
The system stores records in a file named hospital_records.txt. On startup, records are loaded from this file, and any modifications are saved back to it upon exit.
